貸切風呂のご案内

一の湯(金泉・銀泉)

大きな銀泉湯船と丸型金泉湯船を備えております。積み上げた岩の野趣と里山をお楽しみください。7～8名様まで一度にご入浴いただけます。

二の湯(金泉・銀泉)

心安らぐ苔の庭園を配し、大きな銀泉湯船と四角の金泉湯船が特徴です。7～8名様まで一度にご入浴いただけご家族でのご利用にもおすすめです。

三の湯(金泉・銀泉)

四角い銀泉と、同じく四角い金泉湯船は2～3名様向けのお風呂です。ほどよいサイズの湯船で、カップルやおひとりでのご利用におすすめです。

四の湯(金泉)

丸い金泉湯船と銀泉湯船を準備しております。裏山の緑がすぐ近くの森林浴効果も抜群です。ご夫婦やカップルで是非ご利用くださいませ。

五の湯(銀泉)

八角形の形が特徴の銀泉の湯船は4～5名様でお入いただけます。ぜひご家族やお友達同士で輪になってお楽しみください。

六の湯(金泉・銀泉)

四角い金泉と銀泉の湯船を備えております。4～5名様ご入浴いただけますので、ご家族でのご利用にお勧めです。

七の湯(金泉)

金泉と銀泉の湯船を備えており、2～3名様向けのお風呂です。ご夫婦やカップルでぜひご利用くださいませ。

八の湯(金泉)

四角い湯船には金泉のお湯が備わっており4名様程でのご入浴がお勧めです。ぜひ金泉のお湯をご満悦くださいませ。

五の湯から八の湯をご予約の際はフロントまでお問い合わせください
